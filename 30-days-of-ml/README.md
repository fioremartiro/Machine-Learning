# 30 Days of Machine Learning Challenge

Welcome to my 30-day Machine Learning challenge. This track focuses on building practical, production-ready code with a strong emphasis on **Finance** and **Healthcare** applications.

- **Goal:** Build habit, technical depth, and muscle memory.
- **Commitment:** 1 to 2 hours daily.
