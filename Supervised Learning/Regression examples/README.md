# Regression Algorithms in Supervised Learning

In this section, we explore regression algorithms whose purpose is to find the mathematical formula (represented as a line or curve) that best represents the data's trend to predict exact numerical magnitudes.

---

## 1. Linear Regression
The most fundamental form of regression that connects one input (X) with one output (Y) via a straight line, predicting constant and proportional changes.

## 2. Multiple Regression
An evolution that considers several input variables simultaneously to predict a single result, understanding how multiple causes interact at the same time.

## 3. Polynomial Regression
Used when the data relationship requires a curve to be accurate, allowing the model to capture trends that grow or decrease at varying speeds.

## 4. Ridge Regression
A version designed to prevent the model from becoming overly complex; it adds a penalty that makes the model more stable when dealing with noisy data.

## 5. Lasso Regression
Functions as an intelligent feature selector; it applies a strict penalty that can discard irrelevant variables, leaving only the data that truly impacts the result.

## 6. Elastic Net Regression
A hybrid model that combines the strengths of both Ridge and Lasso. It is the most robust tool for situations where many input variables are highly correlated with each other.
