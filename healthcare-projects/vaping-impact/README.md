# Vaping Impact Visualizer

A modern, interactive web application designed to visualize the health and financial impact of vaping. This tool helps users understand their consumption habits through data visualization and provides a shareable "Social Card" to raise awareness.

## Features

*   **Interactive Dashboard:** Input your daily vaping habits to see immediate stats.
*   **Health Impact Visualization:** Visual representation of lung health based on usage.
*   **Financial Calculator:** Estimates monthly and yearly costs associated with vaping.
*   **Social Sharing:** Generate a sleek, shareable image of your impact statistics.
*   **Dark Minimalist Design:** A premium, dark-themed UI focused on readability and aesthetics.

## Tech Stack

*   **Frontend:** React (Vite)
*   **Styling:** Vanilla CSS (Dark Minimalist Theme)
*   **Libraries:** `html2canvas` for image generation.

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## Project Structure

*   `src/components/`: Reusable UI components (InputForm, Visualizer, SocialCard).
*   `src/App.jsx`: Main application logic.
*   `src/index.css`: Global styles and theme variables.
