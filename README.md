# Machine Learning

This repository is dedicated to sharing machine learning projects, tutorials and step-by-step exercises. Here you’ll find a growing collection of examples and real-world projects, organized by topic and technique.

## What you’ll find here

- **[Supervised Learning](./Supervised%20Learning):** The core of the repository, containing detailed examples of:
    - **Regression:** Predicting magnitudes (How much?).
    - **Classification:** Predicting identities (What is it?).
- **[Healthcare Projects](./healthcare-projects):** Real-world machine learning projects focused on healthcare data and applications.
- **Deep Learning (Coming Soon):** Examples and mini-projects using neural networks and advanced deep learning techniques.
- **And more:** I’ll be adding new sections over time, including unsupervised learning (clustering), dimensionality reduction and data preprocessing.
