### 🩺 Applied Healthcare AI & Web Apps
- **Directory:** `./healthcare-projects`
- **Featured Projects:**
  - **Cardiology AI Assistant:** RAG Chatbot using Gemini 2.0, Pinecone, and FastAPI.

  - **Parkinson's Disease Detection:** Voice analysis using SVM.

- **Status:** Ongoing 🟡
